删除分组

1、请求地址：/api/deleteGroupById

2、请求方式：POST

3、请求参数

参数 类型 必须 说明
id int 是 数据id

4、返回参数

无

5、请求示例

POST {{baseUrl}}/api/deleteGroupById
Content-Type: application/json
X-Token: <token>

{
  "id": 1
}

6、返回示例

{
  "code": 0,
  "data": null,
  "msg": "success"
}
